## Project Overview
This is a **UX/UI design prototype** built for the Fabrika assignment.  
It demonstrates a simple, modern, and intuitive web app that allows **small businesses** to create and share digital coupons with their customers.  

The flow is designed to be **fast, frictionless, and mobile-friendly**, making it easy for caf√© owners, boutiques, and small shops to engage their audience.

---

## Features
- **Login with Business Info** ‚Äì simple email-based login flow.  
- **Create Coupons** ‚Äì add title, description, discount type, expiry date, and terms.  
- **Live Preview** ‚Äì see coupon updates instantly while creating.  
- **Share Flow** ‚Äì generate a link for WhatsApp, Telegram, or direct copy.  
- **Edit & Delete** ‚Äì manage existing coupons from the dashboard.  
- **Duplicate Check** ‚Äì prevents creating identical coupons accidentally.  
- **Basic Metrics** ‚Äì quick stats for views, redemptions, and active coupons.
